# Devopensource_PortugalRegions
Magento module for create regions of country portugal.

## Features
Create regions for portugal.

## Requirements

* Magento 1.4 - 1.7
* Modgit installerd

## How to Install

* modgit init .
* modgit clone git@github.com:DevopensourceTeam/TaxVatNumber.git
